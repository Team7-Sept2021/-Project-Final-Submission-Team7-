#Installation


Requirements
* Pandas
* Numpy
* Matplotlib
* Seaborn
* Sklearn


#Project:
The project is the creation of a regression model for predicting the mpg value.


#Objective:
1. Identifying the problem.
2. Exploratory Data Analysis (EDA).
3. Pre-Processing the Data, so that they can be used for predicting the mpg value.
4. Build, train and evaluate a Regression model to predict the miles per gallon (mpg).
